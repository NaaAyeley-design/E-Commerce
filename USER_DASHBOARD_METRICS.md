## üìä ORDER METRICS

### Basic Order Statistics
1. **Total Orders** - Count of all orders (lifetime)
2. **Total Orders This Month** - Count of orders in current month
3. **Total Orders This Year** - Count of orders in current year
4. **Orders by Status** - Breakdown by status (pending, processing, shipped, delivered, cancelled, completed)
5. **Average Order Value** - Total spent / Total orders
6. **Largest Order** - Highest single order amount

8. **Order Frequency** - Average days between orders

## üìù NOTES

- All metrics should be calculated from actual database data
- Use proper date filtering (timezone-aware)
- Handle edge cases (no orders, empty cart, etc.)
- Cache expensive calculations for performance
- Update metrics in real-time or near real-time
- Provide tooltips explaining each metric
- Allow date range filtering for trend metrics
- Make metrics interactive (click to drill down)
