# KenteKart Web App Color Palette

## Primary Brand Colors

### Core Colors (Refined Minimal Palette)
- **White**: `#FFFFFF` - Pure white backgrounds
- **Warm Beige**: `#F4EDE4` - Light background tones
- **Terracotta**: `#C67D5C` - Primary accent color (buttons, links, borders)
- **Deep Brown**: `#8B6F47` - Secondary accent, hover states
- **Dark Brown**: `#5C4A3A` - Darker accents
- **Text Dark**: `#3A2F26` - Primary text color
- **Text Light**: `#8B7F74` - Secondary text, placeholders

## Legacy/Accent Colors (Hero Sections & Special Elements)

- **Ochre**: `#CC8B3C` - Warm golden tone
- **Rust**: `#B7410E` - Deep reddish-brown
- **Amber**: `#FF9A56` - Bright orange-amber
- **Sand**: `#E3B778` - Light sandy beige
- **Burnt Sienna**: `#E97451` - Orange-brown
- **Savanna Gold**: `#D4A574` - Golden tan
- **Earth**: `#6B4423` - Dark earth tone
- **Cream**: `#FFF8E7` - Light cream background

## CSS Variable Names

```css

  --white: #FFFFFF;
  --warm-beige: #F4EDE4;

  --deep-brown: #8B6F47;
```

## Color Usage Guidelines

### Primary Actions
- Use **Terracotta** (`#C67D5C`) for primary buttons and links
- Use **Deep Brown** (`#8B6F47`) for hover states

### Text
- Use **Text Dark** (`#3A2F26`) for primary text
- Use **Text Light** (`#8B7F74`) for secondary text and placeholders

### Backgrounds
- Use **White** (`#FFFFFF`) for main content areas
- Use **Warm Beige** (`#F4EDE4`) for subtle background sections
- Use **Cream** (`#FFF8E7`) for special sections

### Borders & Dividers
- Use **Terracotta** with opacity (0.15-0.2) for subtle borders
- Use solid **Terracotta** for active states and emphasis
